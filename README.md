# TechMe There - Portfolio Site

Next.js + Tailwind CSS portfolio for Justin Yamini

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment to Vercel

1. Create a new GitHub repo: `jyamini26/techmethere`
2. Push this code to GitHub
3. Go to Vercel.com → Import project → Select GitHub repo
4. Deploy
5. Point domain in GoDaddy DNS settings

## Pages

- `/` - Home
- `/about` - About & Bio
- `/services` - Syzen.ai Services
- `/portfolio` - Case Studies
- `/respond-edu` - Education Platform
- `/blog` - Blog Articles
- `/contact` - Contact Form
